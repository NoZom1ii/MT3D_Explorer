# MT3D_Explorer
>利用ceisum实现modEM反演数据的三维演示
>目前可用版本为3dmt4.0.html


核心功能
点位数据显示（支持导入大地电磁测点集群 .dat数据）
反演网格显示 （支持导入反演网格数据并筛选核心区网格 .ws数据）
地下电阻率模型显示 （支持导入多个电阻率数据文件实现多层显示电阻率模型 .xyz文件）
电阻率筛选 （可以通过预输入电阻率值筛选特定的模型）
深度筛选 （可以通过预输入深度去除特定深度的电阻率模型）



操作流程
需要准备点位数据文件.dat  网格数据文件.ws  电阻率数据文件.xyz多个
1.导入点位 “选择文件”选择.dat文件后 可输入高程 高程为测点的纵向高程 默认为5500以青藏高原平均海拔为例 后勾选“显示点位”即可实现测点的实时显示 可以通过点击高程步长按钮上下移动点位

2.导入网格 “选择文件”选择.ws后 需要在三维电阻率模型“选择文件”中选择至少一个.xyz文件 后勾选“显示核心区网格”即可实现反演的核心区网格的实时显示

3.三维电阻率模型显示 需先导入网格选择.ws文件后可选择多个.xyz文件  之后预输入电阻率筛选，推荐选择30 点击“显示地下电阻率模型” 等待模型加载 加载完毕后拖动右上角“地球透明度”于100以下即可看到地下的三维电阻率模型  预输入深度筛选例如10 后点击“应用深度筛选”即可隐藏对应深度的地下单元体
点击电阻率筛选的步长按钮也可以实现实时显示最新筛选值的电阻率模型
点击右下角色度条可以选择最大值 推荐选择100方便观察高导体变化
可在test目录文件中利用自带的文件进行测试


界面功能说明
控制面板  可折叠开放
地球透明度  右上角滑块，调节地表透明度，便于观察
电阻率色度条 右下角 可选择最大值
home键 cesium自带默认键（右上角靠右边）点击可将视角转到 西安上空 方便观察国内全境 新添加的home键（右上角靠左边的）点击可将视角转到核心区网格正上空 方便观察模型



技术栈
Cesium  三维地球渲染
天地图  影像底图
JavaScript  数据交互
HTML5 + CSS3  用户界面与样式设计
